# ZombieTD

Discontinued project which was a tower defense game where monsters would chase after the player. Project was from 2020 - 2021.

The map would be customizable and altered how the pathfinding of the monsters would behave.

Now a tech demo of Godot game engine and using Flow Fields to perform pathfinding at large quantities.

## Instructions

- W,A,S,D keys to move
- Arrow keys to shoot
- B key during "prep" period to build walls

## Requirements

- Godot Version `3.4.2`
